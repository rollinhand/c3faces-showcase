About

C3Faces is C3.js based JSF framework. Its API can be used to create C3.js charts with dynamically changing complex visual animations. It also contains out of the box set of components to start rendering these charts immediately.

Core features are:

  • With JSF declarative way you can create complex charts and define their visual attributes in a really simple manner.
  • Supported are all the main JSF features. Like ajax, value change listeners, converters, etc.
  • Compatible with today's most popular JSF framework, Primefaces.

See examples in the left main menu to see C3Faces in action!

Get it!

Latest version of C3Faces is currently 1.3.

You can easily add C3Faces to your projects Maven dependencies:

<dependency>
    <groupId>org.kivio</groupId>
    <artifactId>C3Faces</artifactId>
    <version>1.4</version>
</dependency>

See project's GitHub page for download, issues, documentation, etc.
